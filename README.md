# AV Car Auto Center

O **AV Car Auto Center** é um sistema desktop desenvolvido para auxiliar no gerenciamento de uma oficina mecânica.  
O sistema permite realizar o cadastro e gerenciamento de clientes, veículos, colaboradores, serviços e ordens de serviço.

## Tecnologias Utilizadas

- **Java** — Linguagem principal do projeto.
- **Java Swing** — Utilizado para criar a interface gráfica.
- **JDBC** — Utilizado para conectar o sistema ao banco de dados.
- **MySQL** — Banco de dados utilizado para armazenar as informações.
- **Maven** — Utilizado para gerenciar as dependências do projeto.

## Funcionalidades

- Cadastro de clientes.
- Cadastro de veículos.
- Cadastro de colaboradores.
- Cadastro de funções.
- Controle de serviços.
- Controle de ordens de serviço.
- Histórico de proprietários dos veículos.
- Armazenamento das informações no banco de dados MySQL.

## Objetivo

Este projeto foi desenvolvido com o objetivo de aplicar na prática os conhecimentos adquiridos durante a graduação em **Análise e Desenvolvimento de Sistemas**, utilizando conceitos de **Programação Orientada a Objetos**, **banco de dados** e **desenvolvimento de aplicações desktop em Java**.
